PROGRAMACION WEB 1 - UNAD ECBTI

INFORMACION GENERAL DEL CURSO

CAMPO DE FORMACIÓN: Disciplinar específico.
TIPO DE CURSO: Metodológico: es teorico - practico, Electivo 2: segundo electivo disciplinar a cursar en el enfasis 3, Enfasis 3: programador experto web.
NÚMERO DE CREDITOS: Tres (3) créditos: con tres unidades didacticas.
CONOCIMIENTOS PREVIOS: Lógica matematica, Lógica para la solución de problemas, Lógica de programación, Análisis de requisitos, Malla curricular de la Tecnología en desarrollo de software, Introducción a la programación web

PROPOSITOS DE LAS UNIDADES DEL CURSO

UNIDAD 1 - INTRODUCCIÓN A LA PROGRAMACIÓN WEB: Comprender las generalidades y fundamentos o conceptos basicos, relacionados con la introducción a la programación web y la introducción al uso de DBMS.
UNIDAD 2 - PROGRAMACIÓN WEB BASICA: Conocer y relacionar la descarga, instalación y manejo del PHP, ASP Y JSP de forma básica, crear elementos de interactividad entre el usuario y pagina web a traves de una arquitectura cliente/servidor.
UNIDAD 3 - PROGRAMACIÓN WEB AVANZADA: Reconocer y hacer uso del lenguaje PHP, ASP Y JSP, en su conceptualización básica, instalar y configurar, asi como manipular datos a traves de DBMS.
